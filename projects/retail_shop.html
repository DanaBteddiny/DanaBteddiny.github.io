Retail Shop

Overview

This project is a C++-based retail shop application designed to efficiently track and manage inventory. It allows store staff to add new items, set unit prices, and update stock quantities. The system also keeps track of sales, ensuring the store manager has an accurate view of available inventory and financial transactions.

Functionality & Workflow

    User Authentication Each staff member logs in using a unique username and password to access their shift
    Inventory Management Users can add new items, update quantities, and check stock levels
    Sales Traking When an item is sold, the system updates stock levels automatically
    Shift-Based Monitoring Sales and expenses are tracked for each shift, allowing staff to monitor their individual sales, income, and stock costs
    Daily Summary At the end of each shift, the system generates a financial report displaying the total revenue and stock purchases during that period
