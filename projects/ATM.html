ATM Application

Overview

This project is a Java-based ATM machine application that allows users to securely perform essential banking transactions. The system enables users to check their balance, withdraw money, deposit funds, and change their PIN through a simple and interactive interface. User authentication is enforced through a secure PIN system, ensuring only authorized users can access their accounts.

Functionality & Workflow

    User Authentication Customers log in using their ATM card number and PIN for secure access.
    Balance Inquiry Users can check their account balance.
    Cash Withdrawal The system verifies the account balance before allowing withdrawals.
    Deposit Functionality Users can deposit money, and the system updates their balance accordingly.
    PIN Management Users can change their PIN for security purposes.
    Transaction Logs A record of transactions is maintained to track deposits and withdrawals.

System Workflow

The ATM system follows a structured process to ensure secure and efficient transactions. When a user inserts their card number and enters their PIN, the system verifies the credentials before granting access. Once authenticated, the user can choose from various banking operations, such as checking their balance, withdrawing money, depositing funds, or changing their PIN. Each transaction is processed securely, ensuring that withdrawals do not exceed the available balance and that deposits are correctly updated. After completing an operation, the system provides a transaction summary before logging the user out to maintain security.
