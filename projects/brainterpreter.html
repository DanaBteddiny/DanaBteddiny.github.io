Brainterpreter

What Is Brainterpreter?

Brainterpreter is an innovative smart home automation system designed to enable users to control appliances using their brainwaves. It provides a hands-free solution for severely disabled individuals, offering intuitive control of household devices through mental commands.

The system uses a combination of brainwave detection and wireless communication to create a seamless and intuitive control interface that adapts to the userâ€™s needs.

How It Works

The system uses the Emotiv Insight headset to detect and extract raw EEG data, which are then sent to a Raspberry Pi via OSC. The Raspberry Pi processes the brainwave data and sends commands to a master ESP32 microcontroller via I2C. These commands control smart plugs connected to appliances, allowing users to turn devices on and off using only their mental commands.

Materials and Methods Used.

Materials Used

    Emotiv Insight Headset
    Raspberry Pi
    ESP32 Chip
    Relays
    Smart Plugs
    SMD Components

Communication Protocols

  • BCI-OSC Used to send brainwave data from the EEG headset to the Raspberry Pi
  • I2C Used to send the commands from the Raspberry Pi to the master ESP32 microcontroller
  • ESP-NOW Used to control the slave ESP32 microcontrollers
