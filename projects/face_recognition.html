Face Recognition

Overview

This project is a Python-based face recognition system that enables users to securely access their saved credentials through facial authentication. Instead of manually entering passwords, users can log in by scanning their face, making authentication more convenient and secure.

Authentication Process

The system captures and processes a userâ€™s facial image using OpenCV and the Face Recognition library. When a user attempts to log in, their live image is compared to the stored dataset. If a match is found, the system retrieves and displays the saved credentials securely. Additional security measures ensure unauthorized users cannot access sensitive information.
